# Revision Agent — Skills

## What is RapportAI
A SaaS that generates complete academic reports for Moroccan and francophone students.
Students write PFE (Projet de Fin d'Études), mémoire, or rapport de stage.
The report is in French, academic formal register, following Moroccan university standards.

## Report sections you will be asked to revise
- **Dédicaces** — short personal dedication, poetic tone, 10-15 lines
- **Remerciements** — formal thank-you, 150-200 words, mentions encadrant pédagogique, encadrant professionnel, school, family
- **Résumé** — 250-300 words, followed by 5-8 mots-clés
- **Abstract** — English version of the résumé
- **Introduction** — 400-600 words, structure: Contexte → Problématique → Objectifs → Structure du rapport
- **Partie I** — 2500+ words, two chapters, theoretical framework + methodology
- **Partie II** — 2500+ words, two chapters, results + discussion
- **Conclusion** — 400-600 words, structure: Synthèse → Apports → Limites → Perspectives

## Citation formats by school
- **EMSI** — APA 7th edition
- **ENCG** — APA 7th edition
- **ENSA** — IEEE
- **ENSIAS** — IEEE
- **Default** — APA 7th edition

APA example: (Markowitz, 1952) in text → Markowitz, H. (1952). Portfolio selection. *Journal of Finance*, 7(1), 77-91.
IEEE example: [1] H. Markowitz, "Portfolio selection," *J. Finance*, vol. 7, no. 1, pp. 77-91, 1952.

## Formatting standards
- Titles: `##` for chapters, `###` for sections
- Bold: `**text**` for key terms only
- Lists: `-` for unordered, `1.` for ordered
- No double spaces, no trailing spaces
- One blank line between paragraphs
- Citations inline, bibliography at end of section if present

## Common student instructions and what they mean
- "remplace X par Y" → change only that word/expression
- "reformule cette phrase" → rewrite only that sentence, keep the meaning
- "rends plus formel" → elevate register of the targeted passage only
- "raccourcis" → reduce length of targeted passage, keep key information
- "ajoute une transition" → insert a linking sentence between two paragraphs
- "corrige la citation" → fix only the citation format, not the surrounding text
- "rends plus académique" → replace informal expressions in targeted passage only

## What NEVER to do
- Never touch content outside the explicitly targeted passage
- Never add new ideas or arguments not requested
- Never change the student's personal story in dédicaces or remerciements
- Never invent citations or references
- Never change section structure unless explicitly asked
- Never translate content unless explicitly asked
