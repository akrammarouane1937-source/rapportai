# RapportAI — Full Build Prompt for Replit Agent (v2 — Updated)

## What You Are Building

A SaaS web application called **RapportAI** that takes a Moroccan student's inputs through a multi-step form and generates a complete, formatted, submission-ready academic report (PFE / rapport de stage / mémoire) as a Word .docx file using the Claude API. Every section is written in professional French academic language matching Moroccan university standards.

**Core value proposition:** 3 mois de rédaction → 30 minutes avec RapportAI.

**Freemium model:** Students go through the entire flow for free (Page de garde, Dédicaces, Remerciements, Sommaire). When Partie I starts generating, it streams 2–3 pages then blurs out and a paywall modal appears with the 3 plans. They've already seen quality → conversion is very high.

---

## Tech Stack

- **Frontend:** Next.js 14 (App Router) + Tailwind CSS + ShadCN UI
- **Auth:** Clerk (email + Google OAuth)
- **Backend:** Next.js API routes
- **AI:** Claude API — `claude-sonnet-4-6` for generation, `claude-haiku-4-5-20251001` for fast tasks
- **Word export:** `docx` npm package (real .docx files)
- **Payment:** Stripe (one-time payment per report, not subscription) — use STRIPE_SECRET_KEY env var directly, no Replit integration needed
- **Database:** PostgreSQL via Neon + Drizzle ORM
- **File storage:** Uploadthing (PDF/Excel/CSV uploads)
- **Charts:** Chart.js (render on Canvas → capture PNG → embed in .docx)
- **CSV parsing:** papaparse + xlsx

## Database Schema

```typescript
export const users = pgTable('users', {
  id: text('id').primaryKey(),           // Clerk user ID
  email: text('email').notNull(),
  plan: text('plan').default('free'),    // 'free' | 'essentiel' | 'pro' | 'premium'
  createdAt: timestamp('created_at').defaultNow(),
})

export const reports = pgTable('reports', {
  id: uuid('id').defaultRandom().primaryKey(),
  userId: text('user_id').references(() => users.id),
  title: text('title').notNull(),
  type: text('type').notNull(),          // 'pfe' | 'rapport_stage' | 'memoire'
  status: text('status').default('draft'),
  currentStep: integer('current_step').default(1),
  completedSteps: integer('completed_steps').array().default([]),
  isPaid: boolean('is_paid').default(false),
  plan: text('plan'),
  pageCount: integer('page_count').default(0),
  revisionCount: integer('revision_count').default(0),
  createdAt: timestamp('created_at').defaultNow(),
  updatedAt: timestamp('updated_at').defaultNow(),
})

export const reportData = pgTable('report_data', {
  id: uuid('id').defaultRandom().primaryKey(),
  reportId: uuid('report_id').references(() => reports.id),
  step: integer('step').notNull(),
  data: jsonb('data').notNull(),
  generatedContent: text('generated_content'),
  wordCount: integer('word_count'),
  updatedAt: timestamp('updated_at').defaultNow(),
})

export const payments = pgTable('payments', {
  id: uuid('id').defaultRandom().primaryKey(),
  userId: text('user_id').references(() => users.id),
  reportId: uuid('report_id').references(() => reports.id),
  stripePaymentId: text('stripe_payment_id'),
  amount: integer('amount').notNull(),   // MAD
  plan: text('plan').notNull(),
  previousPlan: text('previous_plan'),   // for upgrade tracking
  status: text('status').default('pending'),
  createdAt: timestamp('created_at').defaultNow(),
})

export const bibliotheque = pgTable('bibliotheque', {
  id: uuid('id').defaultRandom().primaryKey(),
  userId: text('user_id').references(() => users.id),
  reportId: uuid('report_id').references(() => reports.id),
  fileName: text('file_name').notNull(),
  fileUrl: text('file_url').notNull(),
  extractedText: text('extracted_text'),
  authors: text('authors'),
  year: text('year'),
  title: text('title'),
  doi: text('doi'),
  citationStyle: text('citation_style').default('apa'),
  createdAt: timestamp('created_at').defaultNow(),
})
```

## AI Integration — Claude API

### Client (`lib/claude.ts`)
```typescript
import Anthropic from '@anthropic-ai/sdk'

const client = new Anthropic({ apiKey: process.env.ANTHROPIC_API_KEY })

export async function generateWithCache(
  systemPrompt: string,
  userMessage: string,
  model: 'claude-sonnet-4-6' | 'claude-haiku-4-5-20251001' = 'claude-sonnet-4-6'
) {
  return client.messages.create({
    model,
    max_tokens: 4096,
    system: [{
      type: 'text',
      text: systemPrompt,
      cache_control: { type: 'ephemeral' },
    }],
    messages: [{ role: 'user', content: userMessage }],
    stream: true,
  })
}
```

### .docx Text Formatting (CRITICAL)
Claude must output text with markers for the docx parser:

```
## 1.1 Titre de section

Paragraph text here. Full paragraph, multiple sentences.

## 1.2 Section suivante

Next paragraph here.

### 1.2.1 Sous-section

Sub-section content here.
```

The docx generator MUST:
- Lines starting with `##` → Heading 2 style
- Lines starting with `###` → Heading 3 style
- Empty lines → new Paragraph object (never concatenate into one paragraph)
- Each paragraph = separate `new Paragraph({...})` in docx package
- Body: Times New Roman 12pt, justified, 1.5 line spacing
- 12pt space before each Heading 2, 6pt after each paragraph

### Context Accumulation
Every generation call includes ALL previous steps' data:
- Partie I: theme + école + filière + keywords + problématique + résumé + chapitres + sources
- Partie II: everything from Partie I + Partie I generated content summary + uploaded CSV data
